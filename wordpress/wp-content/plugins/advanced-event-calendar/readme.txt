Advanced Event Calendar wordpress plugin for displaying events on a full-sized calendar. This plugin can embed Google calendar easily.


== Description ==

Advanced Event Calendar wordpress plugin will show event calendar with the best and beautifull way on your site. Its fully responsive plugin.

Also you can add your event in wordpress & show in calendar format. This plugin as awesome lightbox effect for each event.

This plugin has super easy shortcode builder so you can easily embed any spreadsheed in your wordpress editor.

This plugin is totally wordpress shortcode powered. So, you can embed calendar anywhere like page, post, widget also in your theme's php files.


See live demo here: http://demo.wpeffects.com/advanced-event-calendar/

<h2>Events form google calendar</h2>

<p><strong>You must first have a Google Calendar API Key:</strong></p>

<ol>
	<li>Go to the Google Developer Console and create a new project (it might take a second).</li>
	<li>Once in the project, go to APIs & auth > APIs on the sidebar.</li>
	<li>Find "Calendar API" in the list and turn it ON.</li>
	<li>On the sidebar, click APIs & auth > Credentials.</li>
	<li>In the "Public API access" section, click "Create new Key".</li>
	<li>Choose "Browser key".</li>
	<li>If you known what internet domains will host your calendar, enter them into the box. Otherwise, leave it blank. You can always change it later.</li>
	<li>Your new API key will appear. It might take second or two before it starts working.</li>
</ol>

<p><strong>Make your Google Calendar public:</strong></p>

<ol>
	<li>In the Google Calendar interface, locate the "My calendars" area on the left.</li>
	<li>Hover over the calendar you need and click the downward arrow.</li>
	<li>A menu will appear. Click "Share this Calendar".</li>
	<li>Check "Make this calendar public".</li>
	<li>Make sure "Share only my free/busy information" is unchecked.</li>
	<li>Click "Save".</li>
</ol>

<p><strong>Obtain your Google Calendar's ID:</strong></p>

<ol>
	<li>In the Google Calendar interface, locate the "My calendars" area on the left.</li>
	<li>Hover over the calendar you need and click the downward arrow.</li>
	<li>A menu will appear. Click "Calendar settings".</li>
	<li>In the "Calendar Address" section of the screen, you will see your Calendar ID. It will look something like "abcd1234@group.calendar.google.com".</li>
</ol>


<p>Go to <a href="http://google.com/calendar">Google Calendar</a>. Create new calendar form menu. </p>
<img src="assets/images/Screenshot_31.png" alt="" />
<p>Give a calendar title & <strong>Check</strong> Make this calendar public</p>
<img src="assets/images/Screenshot_32.png" alt="" />

<p>Then click on <strong>Create Calender</strong>. You will see an warning. Click yes.</p>

<p>Now add some event. After adding event, Go go calendar settings again. From Calendar Address, click on XML badge. You will see a link. Copy that link. </p>


<p>Now, you are ready to embed google event calender</p>

<h3>Embedding event in page or post</h3>

<p>After install, the plugin will add a button in your post or page editor. See screenshot</p>


<p>If you want to embed event form event custom post click on Insert event form event category. Then type your category name that you are created when you added events.</p>

<p>Now click ok, you will get event shortcode.</p>

<p>If you want to embed event form your google calendar, click on Insert event form google calendar. Then paste your google calendar link you collected. </p>


<p>Now click ok, you will get event shortcode.</p>

== Installation ==

Installing this plugin as regular wordpress plugin.

After install and active, You will see a custom post called <strong>Events Calendar</strong> from the dashboard menu. Create a category form Events Calendar > Events Calendar Category. Add some event on this category. Adding event is super easy. Just go on Events Calendar > Add New. Then you will see event editor. Add title, Description & event dates.
To embedding event in page or post just use the shortcode button called <strong>Event Calendar Shortcode</strong> from the editor. Type category name and your are done.
